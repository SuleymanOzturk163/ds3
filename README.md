### 🛡️ THE JOURNEY
Bu platform, **Dark Souls III** evreninin karanlık atmosferini dijital dünyaya taşımak amacıyla geliştirilmiştir. Sadece bir wiki değil; kullanıcıların topluluk oluşturduğu, lore tartışıp strateji paylaştığı dinamik bir **CMS (Content Management System)** projesidir.

---

### 🌑 CORE FEATURES
| Feature | Description |
| :--- | :--- |
| **Lore Archives** | Boss, lokasyon ve ekipman detaylı içerik yönetimi. |
| **Undead Registry** | Güvenli kayıt ve kullanıcı kimlik doğrulama sistemi. |
| **Bonfire Discussion** | Topluluk odaklı yorum ve moderasyon paneli. |
| **Dark Aesthetic** | Souls serisinin ruhuna sadık, modern UI/UX tasarımı. |

---

### 🛠️ TECH STACK
* **Backend:** `ASP.NET Core MVC`
* **Data:** `Entity Framework Core` (Code-First)
* **Frontend:** `CSS3`, `JavaScript`, `HTML5`
* **Tools:** `Visual Studio Code`, `Git/GitHub`
